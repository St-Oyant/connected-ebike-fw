# Connected E-Bike Firmware

Demo repository used to illustrate software governance for a multi-discipline product in 3DEXPERIENCE.

## Product context
This repository represents the controller firmware of an electric bike.

The firmware interacts with:
- torque sensor
- wheel speed sensor
- battery management data
- motor controller

## Functional scope
- pedal assist management
- thermal protection
- battery protection
- walk assist logic

## Demo version
Current reference release: v1.0.0
